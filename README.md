# The Two Alexes

![Cover](cover.png)

A novella by Joshua Szepietowski

---

Two people, both named Alex, are stationed on a platform embedded inside the corona of an unstable star.

One is a stellar physicist.
One is a pilot.

They cannot leave.
They cannot replace each other.
They cannot escape the star or each other.

One Alex is kind, warm, and emotionally self-contained.
The other Alex needs that warmth.

This is not a story about rejection.
It is a story about asymmetry.
